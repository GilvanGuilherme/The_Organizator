# THE ORGANIZATOR 🎯

O **THE ORGANIZATOR** é um sistema inteligente de controle de metas pessoais. Desenvolvido para simplificar a organização da rotina, o projeto permite que usuários planejem, acompanhem e concluam seus objetivos de forma estruturada e intuitiva.

---

## 🚀 Funcionalidades

O sistema oferece uma experiência completa de gestão, dividida em dois eixos principais:

### **1. Gestão de Acesso**

- **Cadastro:** Criação de novos usuários.
- **Login:** Autenticação segura para acesso ao painel pessoal.

### **2. Painel de Controle (Dashboard)**

- **Recuperação Dinâmica:** Listagem de metas integrada diretamente a um banco de dados SQL.
- **CRUD Completo:** Opções para criar, editar ou remover metas existentes.
- **Controle de Status:** Checkbox seletivo para marcação de conclusão **Total** ou **Parcial**.
- **Cronograma:** Definição rigorosa de datas de **Início** e **Conclusão**.

---

## 🛠️ Tecnologias

- **Front-end:** HTML5, CSS3 (Google Fonts Poppins) e JavaScript (Vanilla).
- **Back-end:** Node.js (Integração de APIs).
- **Banco de Dados:** SQL.
- **Versionamento:** Git.
