# THE ORGANIZATOR

O **THE ORGANIZATOR** é um sistema inteligente de controle de metas pessoais. O projeto foi desenvolvido para ajudar usuários a planejar, acompanhar e concluir seus objetivos de forma estruturada, permitindo o registro detalhado de prazos e níveis de entrega (total ou parcial).

---

## 🚀 Funcionalidades

O sistema está dividido em duas etapas principais de interação:

### **Tela 01: Acesso**

- **Cadastro:** Criação de novos usuários.
- **Login:** Autenticação segura para acessar o painel pessoal.

### **Tela 02: Painel de Controle (Boas-vindas)**

- **Recuperação Dinâmica:** Listagem de metas recuperadas diretamente de um banco de dados SQL.
- **Gestão de Registros:** Opções para **Editar** ou **Remover** metas existentes.
- **Controle de Status:** Checkbox seletivo para marcação de conclusão **Total** ou **Parcial**.
- **Cronograma:** Definição de datas específicas de **Início** e **Conclusão**.
